# 表記ルール（STYLE）— 第2章 静電気

本書（電験三種アカデミア理論・第2章）における文体・表記の統一ルール。
校正時は本ファイルと照合し，不一致箇所を「表記ゆれ」として指摘する。

---

## 1. 文体

| 項目 | ルール | 例 |
|------|-------|----|
| 語尾 | 「〜ます」調（丁寧体） | ○「〜帯電します。」／×「〜帯電する。」 |
| 句読点 | 「，」「．」 | 本書は横書き理工書の慣例に従い **「，」「。」** を使用（読点は全角カンマ）。|
| カタカナ語の長音 | 原則つける | ○「コンデンサ」「ベクトル」 |
| 漢数字 vs 算用数字 | 序数は漢字，数量は算用数字 | ○「第2章」「2つの点電荷」「3枚の極板」 |
| 項目箇条書き | `・` または番号 `①②③` | 本書では両方使用 |

## 2. 英数字・記号

| 項目 | ルール | 例 |
|------|-------|----|
| アルファベット | 半角 | ○「V, I, R, Q」 |
| 数字 | 半角 | ○「100」 |
| カッコ | 本文は全角「（）」，数式は通常文中にそのまま | ○「（図2-1）」 |
| カンマ（読点） | **全角「，」** | ○「2つの点電荷 A，B が」 |
| 中点 | 全角「・」 | ○「正・負」 |
| 乗算記号 | `#` ではなく **「×」または「・」** が正しい | ※PDF抽出で `#` は `×` の化け |

## 3. 単位・量記号

| 項目 | ルール | 例 |
|------|-------|----|
| 単位の表記 | **`〔 〕`（全角大カッコ）で囲む** | ○「100〔V〕」「10〔Ω〕」「4×10⁻²〔m²〕」 |
| 数値と単位の間 | **スペースなし** | ○「100〔V〕」／×「100 〔V〕」 |
| 量記号に単位を付ける | 量記号の直後に〔単位〕 | `V〔V〕`, `r〔m〕`, `Q〔C〕`, `F〔N〕` |
| 単位内の除算 | `/` を使用 | `V/m`, `N/C`, `C/m²`, `F/m`, `C²/(Nm²)` |
| 倍数接頭辞 | 大文字・小文字の区別厳守 | ○「μF」「nF」「pC」／×「UF」「NF」 |
| 科学記数 | `×10ⁿ` を使用 | 8.854×10⁻¹² F/m |

## 4. 数式

| 項目 | ルール | 例 |
|------|-------|----|
| 下付き添字 | 小さい数字・文字 | C₁, C₂, εr, ε₀, Fe, VBA |
| 上付き添字 | 指数 | r², a², 10⁻¹² |
| 分数 | 本文中は `分子/分母` or 別行の大型分数 | `F = kQq/r²` または公式ボックス |
| ベクトル量 | 太字 or 矢印。大きさは絶対値記号 `| |` で囲む | 静電力 F はベクトル。大きさ表記は `|F|` や本文中 F |
| 三角関数 | 小文字立体 | sin θ, cos θ, tan θ |
| ギリシャ文字 | 立体（本書は数式組版上） | π, ε, ε₀, εr, ψ, θ |

## 5. 章・節・図表

| 項目 | ルール | 例 |
|------|-------|----|
| 章見出し | 「第◯章」 | 「第2章　静電気」 |
| 節見出し | **「2-1」「2-2」形式**（本書特有：ハイフンでつなぐ） | 「2-1 静電気と静電力」 |
| 図番号 | 「図◯-◯」形式（章-通し番号） | 「図2-1」「図2-10」 |
| 式番号 | 公式ボックス形式（本書は番号付けなし） | — |

## 6. 本書固有の表記パターン

| 要素 | 形式 | 例 |
|------|-----|----|
| 公式ボックス | `◉公式 <タイトル>` | 「公式 クーロンの法則」 |
| メモチェ | `メモチェ！` マーク＋箇条書き | 【用語】説明。 |
| 「〜って何だっけ？」 | 補助解説用のボックス | 「『帯電』って何だっけ？」 |
| キャラのふきだし | 吹き出しで補足（「〜っす！」「〜ですよ！」等のくだけた口調） | 「クーロンの法則は，点電荷どうしの間でだけ使える法則だ！」 |
| NOTE | 問題横の補足 | 「NOTE：公式における距離 r の単位は〔m〕なので…」 |

## 7. 問題・解答・解説

| 要素 | 形式 |
|------|-----|
| 問題番号 | 「問題N」＋出典「オリジナル」「H15（問15）」「R5上（問2）改」 |
| 解説 | 「解説 ▶」または「解説」見出し |
| 解答 | 「解答 ▶ (N)」または「解答 (N)」。数値解は `1.8 N，斥力` のような形 |
| 解図 | 「解図1」「解図2」… |
| 選択肢 | `(1)（2）（3）（4）（5）` （丸数字と半角カッコ混在するが通常は全角丸なし「（1）」） |

## 8. 固有名詞・記号の扱い

| 項目 | ルール | 例 |
|------|-------|----|
| 単位名カナ | 単位記号の横にルビ相当で添える | 〔C クーロン〕〔V ボルト〕〔F ファラド〕 |
| 人名由来単位 | 大文字小文字厳守 | C, V, F, N, J, Wb, Hz |
| SI単位系 | 基本 | — |

## 9. その他

- 記号「±」「×」「÷」などは **全角**
- 矢印「→」は主に因果・変化，「⇔」はほぼ未使用
- 強調は **太字**（本書ではオレンジ系カラー）で表記されることが多い
- **PDF抽出で発生する「×」→「#」「ε」→「f」「π」→「r」「ε₀」→「f0」の化け** は原本の誤記ではない

---

## 10. PDF抽出時の崩れに注意すべきパターン（校正指摘の対象外）

以下は `pdftotext -layout` 抽出時に頻発する崩れで，**原本の誤記ではなく抽出ミス** として扱う：

- `#` → 実際は `×`（乗算記号）
- `f0` → 実際は `ε₀`
- `f` → 実際は `ε`（誘電率）
- `fr` → 実際は `εr`（比誘電率）
- `r` → 実際は `π`（数式内，`4r`, `4rf0` のように）
- `rf0` → 実際は `πε₀`
- `rf` → 実際は `πε`
- `}` → 実際は `ψ`（プサイ，電束の記号）
- `2 rf 2` などの不自然な並び → 実際は `2πr²` や `4πr²` の断片
- 分数の上下分離：式の途中で意図なく改行している
- ベクトル矢印の消失：`→V` が `V` になる
- `^` → 上付きの欠落
- `m]`や`〔m〕`の表記ゆれ → `〔m〕`（全角大カッコ）に統一
