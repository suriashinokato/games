2-5 コンデンサと静電容量

【コンデンサのキホン】

コンデンサとは
図2-29（a）のように2枚の金属板を向かい合わせたものをコンデンサといいます。実用上は図2-29（b）のような薄い箔状の構造をしているものが一般に用いられますが，これを広げたものと考えると良いでしょう。

（図2-29 コンデンサ）（a）平行平板コンデンサ：端子・金属板・金属板・絶縁体（b）実際のコンデンサ：端子・アルミ箔（電極）・絶縁体

コンデンサの金属板のことを極板といい，図2-30のようにそれぞれの極板に電池のプラスとマイナスを接続すると，電圧によって自由電子の移動が起こり，それぞれの極板は等量の正負の電荷が現れます（極板の帯電）。電池のプラス側に接続するコンデンサの極板を正極，電池のマイナス側に接続する極板を負極といいます。

（図2-30 電池による充電）（a）充電前：最初は両極板ともに電気的に中性（b）充電中：導線を通じて電子が移動する（c）充電後：両極板がそれぞれ等量の電荷で正負に帯電する（+Q〔C〕, −Q〔C〕）

コンデンサは正負の電荷によって極板間に電界が生じ，電位差ができます。この電位差が電池の電圧と同じになると自由電子の移動が止まります。その後，電池を取り去っても，電荷は互いの引力によって引き合うため，そのまま極板に蓄えられたままになります。このように，2枚の極板を用いることで等量の正負の電荷を蓄えることができる装置がコンデンサであり，電荷を蓄える過程を充電といいます。

メモチェ：
【コンデンサ】2枚の極板を用いることで電荷を蓄えることのできる装置。
【充電】コンデンサの両極板には必ず，同じ量の正負の電荷が蓄えられる。

静電容量
コンデンサに蓄えられる電荷の大きさを Q〔C〕とすると，これは極板間の電位差 V〔V〕に比例します。よって，比例定数を C として
Q = CV〔C〕
と表すことができます。この比例定数 C のことを，コンデンサの静電容量，もしくは電気容量といいます。静電容量が大きいほど，同じ電位差でも蓄えられる電荷の量が多くなるため，静電容量はコンデンサが電荷をどれだけ蓄えられるかの器の大きさを示す指標といえるでしょう。式から静電容量の単位はクーロン毎ボルト〔C/V〕となりますが，これを静電容量特有の単位として〔F ファラド〕と表します。

公式：コンデンサに蓄えられる電荷
Q = CV
Q：コンデンサに蓄えられる電荷〔C〕，C：静電容量〔C/V〕＝〔F〕，V：極板間の電位差〔V〕

コンデンサに電荷が蓄えられる様子は水槽をイメージすると良いかもしれません。水槽に入れる水を電荷とすると，図2-31のように静電容量は水槽の底面積で，極板間の電位差が水位です。同じ水位であれば底面積が広いほど，また同じ底面積であれば水位が高くなるほど，たくさんの水が貯められます。

=== p.124〜125 ===

（図2-31 コンデンサと水槽）（a）コンデンサ：電位差・電荷量・静電容量 C〔F〕／V〔V〕／+Q, −Q（b）水槽：水量（電荷量）／水位（電位差）／底面積（静電容量）

平行平板コンデンサの静電容量
同じ形状，同じ大きさの2枚の極板を平行に並べてつくるコンデンサを平行平板コンデンサといいます。極板の面積を S〔m²〕，極板間距離を d〔m〕，極板間が真空の平行平板コンデンサの静電容量 C〔F〕は，次の式で求めることができます。

公式：平行平板コンデンサの静電容量
C = ε₀·S/d
C：静電容量〔F〕，ε₀：真空の誘電率〔F/m〕，d：極板間距離〔m〕，S：極板の面積〔m²〕

これまで誘電率の単位は〔C²/(Nm²)〕でしたが，静電容量の単位〔F〕を使うと，公式のように〔F/m〕でも表すことができます。

（図：平行平板コンデンサの静電容量の導出）
平行平板の内部電界 E=Q/(ε₀S)
電界と電位差の関係 E=V/d
V/d = Q/(ε₀S) → Q=ε₀(S/d)V=CV
静電容量の式 C=ε₀S/d
ふきだし：今まで学習してきた電荷と電界，電位差の関係性の知識をフル活用して導くっす！

メモチェ：
【平行平板コンデンサの静電容量】極板の面積及び誘電率に比例し，極板間距離に反比例する。

導体球の静電容量
コンデンサは電荷を蓄える機能をもった装置ですが，コンデンサに限らず導体は一般に電荷を蓄える器としてのはたらきがあります。半径 a〔m〕の導体球の静電容量は，+Q〔C〕の電荷を与えることで次のようにして求めることができます。

（図：導体球の静電容量）
導体球の電位 V=Q/(4πε₀a)
電荷と電位の関係 Q=4πε₀a·V=CV
導体球の静電容量 C=4πε₀a
ふきだし：電荷と静電容量の関係式 Q=CV から，導体球のもつ静電容量を求めることができますよ！

メモチェ：
【導体の静電容量】導体はそのものが電荷を蓄えるはたらきをもっている。

問題10（オリジナル）
極板の面積が 4×10⁻² m²，極板間距離が 0.1 mm のコンデンサについて，次の問いに答えよ。
（1）このコンデンサの静電容量〔nF〕を求めよ。ただし，真空の誘電率は ε₀=8.85×10⁻¹² F/m とする。
（2）このコンデンサに 300 V の直流電圧を印加したときにコンデンサに蓄えられる電荷〔nC〕を求めよ。

=== p.126〜127 ===

（3）電源を接続したまま極板間距離を 0.3 mm にしたとき，電源を通過する電荷量〔μC〕を求めよ。　※原本：〔nC〕→〔μC〕の表記混在に注意

解説
（1）静電容量の式より
C = ε₀·S/d = 8.85×10⁻¹² × (4×10⁻²)/(0.1×10⁻³) = 3.54×10⁻⁹ F = 3.54 nF

（2）コンデンサに蓄えられる電荷 Q〔nC〕は Q=CV から
Q = CV = 3.54×10⁻⁹ × 300 = 1.062×10⁻⁶ = 1.062 μC　※原本表示「1.062 nC」はμCの誤植

（3）極板間距離を3倍するということは，すなわち静電容量を 1/3 にするということになるので，蓄えられる電荷量も 1/3 になります。もともと電荷が溜まっているところから 1/3 まで電荷が減るということは，もとの 2/3 の電荷が電源を通過するということになるので，求める電荷量 Q'〔μC〕は
Q' = (2/3)Q = (2/3)×1.062 = 0.708 μC

NOTE：公式を使って解く基本問題ですが，（3）は問題文の意味をしっかり読み解かないといけません。こういった国語力も電験では求められます。

解答 （1）3.54 nF （2）1.062 μC （3）0.708 μC
※原本：解答表示は（2）1.062 nC／（3）0.708 nC となっており，問題文の単位系と不整合。

問題11（H18 問1）
真空中に半径 6.37×10⁶ m の導体球がある。これの静電容量〔F〕として，最も近いのは次のうちどれか。ただし，真空の誘電率を ε₀=8.85×10⁻¹² F/m とする。
（1）7.08×10⁻⁴
（2）4.45×10⁻³
（3）4.51×10³
（4）5.57×10⁴
（5）1.78×10⁵

NOTE：半径 6.37×10⁶ m の導体球は地球とほぼ同サイズなので，地球を大きな導体球であるとみなしたときの静電容量を求めるという趣旨の問題といえます。

解説 導体球の静電容量 C は，その半径を a〔m〕とすると C=4πε₀a〔F〕で表されます。これに各値を代入すると
C = 4×π×8.85×10⁻¹² × 6.37×10⁶
  ≒ 708.42×10⁻⁶ = 7.08×10⁻⁴ F
解答 （1）

【誘電体を挿入したコンデンサ】

誘電体と誘電分極
誘電体は電気を通すことができない絶縁体の一種です。しかし，あえて絶縁体ではなく誘電体という名前がつけられているのは，誘電体がもつある特性を積極的に使うことを目的としているからです。
絶縁体は自由電子をほとんどもたないため，電界中においても導体のような静電誘導は生じません。しかし，誘電体は電界中に置くことによって，図2-32（a）のように誘電体内の電子が外部電界の影響によって若干引っ張られて一方に偏ります。この偏りを誘電体全体で見ると，導体の静電誘導のように誘電体の表面に正負の電荷が誘導されます（図2-32（c））。これを静電誘導と区別して，誘電分極といいます。

（図2-32 誘電分極）（a）電界により生じる電荷の偏り（b）偏りの連鎖（c）誘電分極

誘電分極も静電誘導と同様に，外部の電界とは逆向きの電界が内部に発生するため，導体のように内部電界を 0 にするまでの誘導は起きませんが，誘電体内部の電界が弱められることになります。
誘電分極のしやすさを表す指標が，誘電率 ε です。したがって，誘電率が大きいほど誘電分極が活発になり，内部の電界を弱めるはたらきが大きくなります（図2-33）。

（図2-33 誘電率と内部電界）（a）誘電率が小さい場合：誘電率が小さいと内部の電界は少しだけ弱まる（b）誘電率が大きい場合：誘電率が大きいと内部の電界はかなり弱まる

誘電体を挿入したコンデンサ
コンデンサの極板間に誘電体を挿入すると，コンデンサの静電容量が大きくなり，絶縁性が良くなってコンデンサの最高使用電圧（これを耐電圧といいます）が高まるため，一般にコンデンサは誘電体を挿入しています。誘電率 ε〔F/m〕の誘電体を挿入したコンデンサの静電容量は，挿入する前の誘電体がない場合の静電容量を C₀ とすると，次の式で求めることができます。

公式：誘電体を挿入したコンデンサの静電容量
C = ε·S/d = εr·ε₀·S/d = εr·C₀
C：静電容量〔F〕，ε：誘電体の誘電率〔F/m〕，d：極板間距離〔m〕，S：極板の面積〔m²〕，εr：誘電体の比誘電率，C₀：誘電体がない場合の静電容量〔F〕

図2-34のように，電圧を加えて電荷が蓄えられたコンデンサに誘電体を挿入すると，誘電体内部の電界が弱められるので，挿入直後はコンデンサ極板間の電位差が V〔V〕よりも小さくなります。すると，電位差が V〔V〕となるようにさらにコンデンサに電荷が蓄積されていきます。そうして，誘電体の内部電界が挿入前と等しくなるまで電荷が増えることで，コンデンサには誘電体を挿入した後の方がより沢山の電荷が蓄えられることになります。以上のことから，誘電体はコンデンサの「電荷を蓄える」という機能を，「誘電分極」という特性を利用して強化することができる材料といえます。また，C と C₀ の比の値 C/C₀ は，挿入する誘電体の種類で決まる定数であり，これを比誘電率 εr といいます。

（図2-34 誘電体を挿入したコンデンサ）（a）誘電体挿入前充電後：充電後は極板間の電位差は電源電圧と同じになる（b）誘電体挿入直後：挿入直後は極板間の電界が弱まり，電位差は V より小さくなる（c）充電後：電位差が電源電圧と同じになるようにさらに電荷を蓄える

問題12（H22 問2）
図に示すように，電極板面積と電極板間隔がそれぞれ同一の2種類の平行平板コンデンサがあり，一方を空気コンデンサ A，他方を固体誘電体（比誘電率 εr=4）が満たされたコンデンサ B とする。両コンデンサにおいて，それぞれ一方の電極に直流電圧 V〔V〕を加え，他方の電極を接地したとき，コンデンサ B の内部電界〔V/m〕及び電極板上に蓄えられた電荷〔C〕はコンデンサ A のそれぞれ何倍となるか。その倍率として，正しいものを組み合わせたのは次のうちどれか。ただし，空気の比誘電率を1とし，コンデンサの端効果は無視できるものとする。
（図：コンデンサ A（空気）と コンデンサ B（εr=4（固体））に電圧 V〔V〕を印加）

　内部電界　電荷
（1）1　　4
（2）4　　4
（3）1/4　　4
（4）4　　1
（5）1　　1

NOTE：各コンデンサ内部の電界は，共通の物理量（本問の場合は電圧及び極板間距離）が何になるかによって電界の比較に用いる式が変わります。蓄えられた電荷が等しい場合の電界の比較は，後の節で学習します。

解説 題意から，ともに印加される電圧 V が等しく，また電極板間隔も等しいことから，V=Ed を変形した E=V/d の式より，コンデンサ A と B とで V も d も等しいため，各コンデンサの電界は等しくなります。したがって，コンデンサ B の A に対する内部電界の倍数は 1 となります。
真空の誘電率を ε₀ とすると，各コンデンサに蓄えられる電荷 QA〔C〕及び QB〔C〕は次式で求められます。
QA = CAV = ε₀(S/d)V
QB = CBV = εr·ε₀(S/d)V = 4ε₀(S/d)V
上式より QB=4QA となるので，QB の QA に対する倍数は
QB/QA = 4
解答 （1）

【静電エネルギー】

コンデンサの放電
図2-35のように電池により充電したコンデンサに，電池を取り去る代わりに電球につなげると，短い時間ですが電流が流れて電球が点灯します。これは，充電されたコンデンサの極板間から導線を通じて電荷が移動，すなわち回路に電流が流れるからです。そして，電流が流れることによって電球は光エネルギーを放出します。これを充電に対して放電といいます。このことから，コンデンサは充電することでエネルギーが蓄えられるといえます。このエネルギーを静電エネルギーといいます。

（図2-35 放電）（a）充電：V〔V〕→（b）電源を取り去った後 →（c）放電：電球点灯

コンデンサに蓄えられる静電エネルギーの求め方
電荷をコンデンサに蓄えるために必要な仕事が静電エネルギーになります。この仕事の大きさを求めてみましょう。図2-36のように，静電容量 C〔F〕のコンデンサの一方の極板から電荷を +1 C ずつもう一方の極板に移動させて，最終的に電荷 Q〔C〕を蓄えて極板間の電位差を V〔V〕にする場合を考えてみます。

（図2-36 電荷を蓄えるために必要な仕事）（a）充電最初：0V（b）充電途中：V/2〔V〕，+Q/2, −Q/2（c）充電最後：V〔V〕，+Q, −Q

最初は電位差が零なので，最初に必要な仕事は 0 J です。しかし，充電の最後は電位差がほとんど V〔V〕なので，+1 C の電荷を移動させるのに必要な仕事もほぼ 1×V=V〔J〕となります。したがって，最初から最後までの仕事を平均すると V/2〔J〕となり，全部で Q〔C〕の電荷を移動させたので，トータルの仕事量を W〔J〕とすると
W = (1/2)V·Q = (1/2)QV〔J〕
となります。この仕事量が，コンデンサに蓄えられる静電エネルギーと等しくなります。この式にコンデンサに蓄えられる電荷の公式 Q=CV を代入することで，静電エネルギーの公式が得られます。

公式：コンデンサに蓄えられる静電エネルギー
U = (1/2)QV = (1/2)CV² = (1/2)·Q²/C
U：静電エネルギー〔J〕，Q：コンデンサに蓄えられた電荷〔C〕，V：極板間の電位差〔V〕，C：コンデンサの静電容量〔F〕

メモチェ：
【静電エネルギー】コンデンサを充電することで蓄えることのできるエネルギー。

問題13（オリジナル）
静電容量 C=1000 μF のコンデンサを充電して U=80 J の静電エネルギーを蓄えたいとき，コンデンサに加える電圧は何〔V〕か。　※原本表記では本文「C=1000 nF」「C=1000 μF」の混在あり

解説 求める電圧を V〔V〕とすると，静電エネルギーの公式より
U = (1/2)CV²
よって
V = √(2U/C) = √(2×80 / (1000×10⁻⁶)) = √(0.16×10⁶) = 0.4×10³ = 400 V
解答 400 V

問題14（H29 問2 改）
極板の面積 S〔m²〕，極板間の距離 d〔m〕の平行板コンデンサ A，極板の面積 2S〔m²〕，極板間の距離 d〔m〕の平行板コンデンサ B 及び極板の面積 S〔m²〕，極板間の距離 2d〔m〕の平行板コンデンサ C がある。各コンデンサは，極板間の電界の強さが同じ値となるようにそれぞれ直流電源で充電されている。各コンデンサに蓄えられる静電エネルギーの総和の値〔J〕を次の（1）〜（5）から一つ選べ。ただし，コンデンサ A に印加している電圧を V〔V〕とし，各コンデンサの極板間の誘電率は ε₀ で同一であり，端効果は無視できるものとする。

（1）3ε₀SV²/(2d)　（2）5ε₀SV²/(4d)　（3）5ε₀SV²/(2d)　（4）9ε₀SV²/(4d)　（5）9ε₀SV²/(2d)

NOTE：ただ式を暗記するのではなく電界と電位差の関係性をコンデンサの構造から理解しましょう。

解説 コンデンサにおいて電界と電位差の間には V=Ed の関係があるため，各コンデンサの電界の強さ E は等しいことから，各コンデンサに印加される電圧 VA〔V〕，VB〔V〕，VC〔V〕は VA=Ed〔V〕，VB=Ed〔V〕，VC=2Ed〔V〕となります。ここで，VA=V とすると，VB=V，VC=2V となります。
A，B 及び C の静電容量 CA〔F〕，CB〔F〕，CC〔F〕は誘電率を ε とすると
CA = εS/d
CB = ε·2S/d = 2CA
CC = εS/(2d) = CA/2
