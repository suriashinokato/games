=== p.79（章扉） ===

第2章　静電気

この章で学ぶこと
静電気とは，電気が動かずに留まっている状態を指します。この章では，電荷の性質やクーロンの法則，電界や電位といった基本概念と，電荷を蓄えるコンデンサについて学びます。電気の「出発点」となる考え方をしっかりと理解することが目的です。

攻略法
公式をやみくもに暗記するのではなく，「プラスとマイナスの電荷のはたらき」をイメージしましょう。力やエネルギーといった，物理の基本や概念をつかむことを優先し，現象を直感的にとらえることが大切です。

=== p.80〜81（2-1 静電気と静電力） ===

2-1 静電気と静電力

【静電気のキホン】

静電気とは
プラスチック製の下敷きなどで頭をこすると，髪の毛が逆立ちます。これは，下敷きと髪の毛との間に摩擦を起こすことによって髪の毛から電子が下敷きに移り，下敷きが負の電気を帯びて，髪の毛が正の電気を帯びたためです。
物質が電気を帯びることを帯電といい，帯電した状態もしくは帯電した電気そのものを静電気といいます。

（図：静電気とは？）摩擦により電子が移動 → 帯電により吸引力が発生
ふきだし：物質が帯電した状態が静電気。冬，ドアノブでパチっとなるときも「静電気だ」っていうよね！

？「帯電」って何だっけ？
原子は正の電気をもつ陽子を含む原子核と負の電気をもつ電子があり，通常は正負の電気の量が等しい状態ですが，ふとしたきっかけで電子が移動して，電子が不足したり過剰になったりすることがあります。この電子の過不足の状態が帯電です。

（図：+11e 陽子の数＝電子の数→電気的に安定。電子が1つ減少。トータルの電荷量が正→正に帯電している状態）

電荷とは
帯電した物質（帯電体という）が帯びている電気のことを，電荷といいます。電荷には正と負の2つの極性があり，正の電気のことを正電荷，負の電気のことを負電荷といいます。電荷が大きいほど，その物質が帯びている電気の量が大きいということです。ただし，「電子は負電荷である」など，電気をもつ粒子のことを電荷ということもあります。

メモチェ：
【正電荷】電子が不足してトータルの電荷量が正の状態。もしくは陽子のもつ電気。
【負電荷】電子が過剰でトータルの電荷量が負の状態。もしくは電子のもつ電気。

電荷の単位
電荷のもつ電気の量を電荷量，もしくは単に電荷（単位：〔C クーロン〕）といいます。電子1粒がもつ電荷量は −1.602×10⁻¹⁹ C であり，これは物質がもちうる電荷量の最小単位です。そして，この電荷量の絶対値を特別に電気素量といい，記号 e で表します。したがって，電子1粒の電荷量は −e〔C〕と表されます。一般に，電荷量をあるまとまった量として記号で表すには Q，もしくは小文字の q が使われます。

=== p.82〜83 ===

（図：電気素量 e と電荷量 Q，q）
【電気素量】電子 −e〔C〕
【電荷量】負電荷 −q〔C〕，正電荷 +q〔C〕
ふきだし：一般に静電力の計算には Q や q が使われますが，電子の運動を考えるときは −e を使います！

メモチェ：
【電荷量 Q，q】あるまとまった量の電荷を表すときに用いる記号。
【電子の電荷量】電子のもつ電気は負であるため，電気素量を使って −e と表す。

静電力
下敷きと髪の毛の例のように，正負それぞれに帯電した物質は互いに引きつけ合う引力がはたらきます。また，正に帯電した物質どうし，負に帯電した物質どうしは互いに反発し合う斥力（せきりょく）がはたらきます。この力のことを静電力といい，静電力の大きさは帯電した電気の量に比例して大きくなります。

（図：静電力）【異符号は引力】【同符号は斥力】
ふきだし：異なる種類の電荷の間には引力，同じ種類の電荷の間では斥力がはたらくっす！

メモチェ：
【異符号の電荷どうし】互いに引きつけ合う引力がはたらく。
【同符号の電荷どうし】互いに反発し合う斥力がはたらく。

？点電荷ってなんですか？
フランス人のクーロンは，下図のようなねじり天秤を用いて，2つの帯電した導体球の間ではたらく静電力の大きさを実験しました。このとき用いた導体球は，空間に対して大きさが無視できるほど小さいとみなせるため，これを点電荷と表現します。また，電子のように電荷をもった粒子のことも点電荷と表現することもあります。
（図：ねじり天秤。振動減衰用円板，帯電した導体球（点電荷）。吊るした導体球の回転量で斥力の大きさを測定する）

【クーロンの法則】

クーロンの法則
2つの点電荷（小さな帯電体）の間で働く静電力の大きさ F〔N〕は，点電荷間の距離 r〔m〕の2乗に反比例し，2つの点電荷のもつ電荷量 Q〔C〕，q〔C〕の絶対値の積に比例するということが実験によって確かめられました。これをクーロンの法則といいます。

=== p.84〜85 ===

（図：クーロンの法則）異符号は引力 / 同符号は反発力 / 電荷量の絶対値の積に比例 / 距離の2乗に反比例
ふきだし：クーロンの法則は，点電荷どうしの間でだけ使える法則だ！

クーロンの法則は次の式で表します。
公式：クーロンの法則
F = k|Q||q|/r² = |Q||q|/(4πεr²)
F：静電力〔N〕，k：比例定数〔Nm²/C²〕，r：2点間の距離〔m〕，|Q|，|q|：電荷量〔C〕，ε：誘電率〔C²/(Nm²)〕

式中の k は，電荷の周りに存在する媒質によってその大きさが変わる比例定数です。実験によって，「真空中で1 C の2つの点電荷が1 m 離れているときに，点電荷に発生する静電力は約 9.0×10⁹ N である」ということが確かめられています。このことから，真空中の場合の比例定数を k₀ とすると，k₀ ≒ 9.0×10⁹ Nm²/C² となり，この真空中での比例定数 k₀ を特にクーロン定数といいます。

クーロン定数の別の表現方法
クーロンの法則における比例定数 k は電磁気学の分野では一般に
k = 1/(4πε)
と表します。ε は誘電率といい，空間を満たしている媒質によって値が変わります。単位は比例定数の逆で〔C²/(Nm²)〕です。空間が真空である場合の誘電率は ε₀=8.854×10⁻¹² C²/(Nm²) であり，このときのクーロン定数 k₀ は
k₀ = 1/(4πε₀)
と表されます。媒質によって変わる誘電率 ε は，便宜のため ε=εrε₀ として真空の誘電率を基準にした比誘電率 εr を用いて表します。代表的な物質として，空気の比誘電率は 1.000586，紙は 2〜2.5，水は約 80 です。比誘電率を用いると k は
k = 1/(4πε) = 1/(4πε₀εr)
となります。以上のことから，クーロンの法則の式は公式にもあるように
F = |Q||q|/(4πεr²)
とも表せます。本書では，クーロンの法則を扱うときは基本的にこの式を使用しますが，問題文でクーロン定数として条件が与えられる場合もありますので，両方の式をしっかり覚えておきましょう。

メモチェ：
【誘電率】周囲の媒質によって値が変わる。真空の誘電率は ε₀=8.854×10⁻¹² C²/(Nm²)

？比誘電率って何のためにあるの？
例えば空間が油などの絶縁体（比誘電率：約3）で満たされている場合は，誘電率は真空の誘電率の約3倍と大きくなり，結果としてそのような空間では静電力は真空の場合の 1/3 と小さくなります。このように，真空中に比べてどの程度静電力が弱まるかを表す指標が比誘電率です。このことは，後で登場する「コンデンサに誘電体を挿入することでコンデンサの性能が変わる」という理論につながります。

複数の点電荷が存在する場合のクーロン力の合成
空間に3つ以上の点電荷が存在する場合，そのうちの1つの点電荷にはたらく力はほかの電荷から受ける静電力を足し合わせたものになります。静電力は向きと大きさをもつベクトルであるため，それぞれの力を足し合わせるときは，ベクトルの和を求める必要があります。このように，それぞれ別々の点電荷どうしではたらく力を重ね合わせることによって，静電力を合成できることを重ね合わせの理といいます。

=== p.86〜87 ===

（図：重ね合わせの理（電荷が3つの場合））AによりCにはたらく静電気力＋BによりCにはたらく静電気力＝Cにはたらく静電気力の総和
ふきだし：それぞれの静電力の大きさが仮に10Nだったとしても，単純に足して20Nとはならないっす！

メモチェ：
【重ね合わせの理】3つ以上の電荷があったとき，ある1つの電荷がほかから受ける静電力は，それぞれの電荷から受ける各静電力のベクトルの和と等しい。

？「ベクトル」って何だっけ？
力など，大きさと向きの2つの量をもつ物理量をベクトルといいます。ベクトルの和は単純な大きさではなく，向きを考慮して足し合わせる必要があります。

問題1（オリジナル）
真空中に2つの点電荷があり，それぞれ q₁=2×10⁻⁷ C，q₂=4×10⁻⁷ C の電荷を持っている。これらの点電荷の距離が2 cm であるとき，点電荷間にはたらく静電力の大きさはいくらになるか，またはたらく力は引力か斥力かを答えよ。ただし，クーロン定数 k₀ は k₀=9.0×10⁹ とする。

解説 題意に沿って図を描くと解図のようになります。
クーロンの法則より，F = k·q₁q₂/r² にそれぞれの値を代入すると
F = 9.0×10⁹ × (2×10⁻⁷ × 4×10⁻⁷)/(2×10⁻²)²
  = 9.0×10⁹ × (8×10⁻¹⁴)/(4×10⁻⁴)
  = 18×10⁻¹ = 1.8 N
q₁ と q₂ は互いに同符号のため，はたらく力は斥力となります。
（解図：2×10⁻⁷ C ←→ 4×10⁻⁷ C，距離 2×10⁻² m，両側に F〔N〕が外向き）

NOTE：公式における距離 r の単位は〔m〕なので，題意で与えられる距離が〔cm〕の場合は 10⁻² を掛けて単位を〔m〕にしてから代入しましょう。

解答 1.8 N，斥力

問題2（オリジナル）
2つの点電荷 A，B が，真空中で互いに5 N の力を及ぼし合っている。点電荷の距離が30 cm で，A の電荷が B の2倍であるとき，B の電荷の大きさ〔C〕はいくらか。ただし，点電荷はともに正電荷とし，クーロン定数 k₀ は k₀=9.0×10⁹ とする。

解説 A の電荷を qA〔C〕，B の電荷を qB〔C〕として題意に沿って図を描くと解図のようになります。
（解図：qA +, qB +，距離 30×10⁻² m，両側に 5 N が外向き）

NOTE：A の電荷を qA とおくなど，自分で量記号を定義して立式できるようになりましょう。

クーロンの法則より F = k·qAqB/r² に，題意から qA=2qB とし，それぞれの値を代入すると
5 = 9.0×10⁹ × (2qB×qB)/(30×10⁻²)² = 9.0×10⁹ × 2qB²/(9×10⁻²) = 2×10¹¹ qB²
qB² = 2.5×10⁻¹¹ = 25×10⁻¹²
∴ qB = 5×10⁻⁶ C

解答 5×10⁻⁶ C

=== p.88〜89 ===

問題3（オリジナル）
図のように xy 平面上に真空中に A，B，C の3つの点電荷があり，それぞれ等しく q〔C〕の電荷をもっている。C 点にある点電荷にはたらく静電力の大きさはいくらになるか。また，はたらく力の向きを答えよ。ただし，クーロン定数を k₀ とする。
（図：A(−a,0), B(0,a), C(0,0) の3つの点電荷が q〔C〕）

解説 C からほかの電荷までの距離はともに a であるため，A と B の電荷から C が受ける静電力の大きさ F〔N〕はそれぞれ等しく
F = k₀·(q·q)/a² = k₀·q²/a² 〔N〕
ただし，力を受ける向きは解図1のようになります。

NOTE：電験では比較的単純な三角比を用いて解く場合が多いので，三角比をしっかり理解しましょう。

直角にそれぞれ等しい力を受けるため，これらの力の合力 F' は解図2のようになり，三平方の定理により右下45°の向きに √2 F の力となります。
したがって，C にはたらく静電力の大きさ F'〔N〕は
F' = √2 F = √2 k₀·q²/a² 〔N〕

解答 √2 k₀·q²/a²〔N〕，紙面右下45°

【導体球にはたらく静電力】

導体球とは？
導体は，主に銅やアルミといった金属などの電気を通しやすい物質です。導体の内部には，原子核の束縛から離れて自由に動き回ることができる自由電子が存在します。
球状の導体のことを導体球といいます。導体の性質についてはのちの節で詳しく紹介しますが，導体球は電荷を与えると，導体内部に電荷がとどまることはなく導体表面に電荷が均等に分布されるという性質をもっています（図2-1）。
（図2-1 導体の静電気的な性質。電荷は内部にとどまらない／導体の表面に電荷が分布する）

点電荷は質量と大きさを無視できるような小さな帯電体ですが，導体球は質量や大きさをもった比較的大きな帯電体です。したがって，電荷をもった2つの導体球が空間にあるときに導体球に発生する力は，静電力のほかに質量によって生じる重力を考慮する必要があります。

天井から吊るした導体球にはたらく力
図2-2のように，質量が m〔kg〕で同じ大きさの導体球 A，B があり，導体球を糸で天井の同じ点から吊るします。2つの導体球にそれぞれ +Q〔C〕の電荷を与えると，電荷が同符号であるため導体球間に斥力 Fe〔N〕がはたらき，離れようとします。しかし，糸で吊るされているため導体球は張力 T〔N〕によって糸に引っ張られます。このとき導体球において静電力 Fe と重力 mg〔N〕とを合成した力 F と，糸による張力 T とがつり合います。
このとき，導体球 A，B の間隔を r〔m〕として空間の誘電率を ε とすると，導体球にはたらく静電力 Fe〔N〕は
Fe = Q²/(4πεr²) 〔N〕
となります。

=== p.90〜91 ===

（図2-2 天井から吊るした導体球にはたらく力）導体球にそれぞれ +Q〔C〕を与える

また，導体球には重力 mg が鉛直下向きに生じており，静電力は水平方向に働いているため，これらを足し合わせた合力 F〔N〕は三平方の定理により
F = √((Q²/(4πεr²))² + (mg)²) 〔N〕
と求めることができ，これが張力 T の大きさと等しくなります。また，図2-3のように糸と鉛直下向きとの間の角度を θ とすると
tan θ = Fe/mg = (Q²/(4πεr²))/mg
という関係があることも覚えておきましょう。

（図2-3 天井から吊るした導体球にはたらく力）同位角であるため同じ角度 / Fe=Q²/(4πεr²) / F=√((Q²/(4πεr²))²+(mg)²) / tan θ = Fe/mg

メモチェ：
【導体球にはたらく力】質量をもった導体球には重力が生じるため，合力は静電力と重力のベクトル和となり，これが糸による張力と等しくなる。

問題4（H15 問15）
真空中において，それぞれ質量 m〔kg〕，電荷 +Q〔C〕の小さな球の帯電体 A 及び B がある。これらの帯電体をそれぞれ長さ r〔m〕の糸で点 P からつるしたところ，図のように，帯電体 A，B の間隔が a〔m〕となって静止した。次の（a）及び（b）に答えよ。ただし，真空の誘電率は ε₀〔F/m〕，重力加速度は g〔m/s²〕とする。また，帯電体 A 及び B の直径は r〔m〕に比べて十分に小さく，糸の質量は無視できるものとする。

（a）帯電体 A，B の間に働く力 F〔N〕の大きさとして，正しいのは次のうちどれか。
（1）Q/(4πε₀a)　（2）Q/(4πε₀a²)　（3）Q²/(4πε₀a)　（4）Q²/(2πε₀a²)　（5）Q²/(4πε₀a²)

（b）帯電体 A，B の静止状態において，糸の鉛直線に対する傾きが θ〔°〕であったときに成立する式として，正しいのは次のうちどれか。
（1）Q²=16πε₀mgr²sin²θ tanθ
（2）Q²=16πε₀mgr²sin²θ/tanθ
（3）Q²=16πε₀mgr²cos³θ/sinθ
（4）Q²=8πε₀mgr²sin²θ tanθ
（5）Q²=8πε₀mgr²sinθ cosθ

解説（a）真空の誘電率 ε₀ が与えられているのでクーロンの法則は F = qAqB/(4πε₀r²) を用います。電荷の大きさはともに +Q〔C〕，電荷間の間隔は題意より a〔m〕なので，両帯電体の間には
F = Q²/(4πε₀a²) 〔N〕
の大きさの力がはたらきます。

NOTE：質量を持った導体球については，このようなつり合いを考える問題が基本問題となります。

（b）質量 m〔kg〕の帯電体にはたらく重力は鉛直下向きに mg〔N〕であり，これに加えて水平方向に（a）で求めた静電力 F〔N〕がはたらくので，静止状態では解図のように表すことができます。静電力と重力の合力のベクトルの向きは糸の向きと同じであるため，解図のように合力と重力とのなす角も θ〔°〕となります。

したがって，次の式が成り立ちます。
tan θ = F/mg = [Q²/(4πε₀a²)]/mg = Q²/(4πε₀a²mg)
∴ Q² = 4πε₀a²mg tan θ　…（1）

解答の選択肢に含まれるのは a ではなく r なので，a を r で表す方法を考えます。
解図より，糸の長さが r〔m〕であり，P 点と B 点との水平距離は a/2〔m〕であるため，三角比より
sin θ = (a/2)/r = a/(2r)
∴ a = 2r sin θ　…（2）

したがって，式（2）を式（1）に代入すると
Q² = 4πε₀(2r sin θ)² mg tan θ = 16πε₀mgr² sin²θ tan θ

解答 （a）-（5），（b）-（1）
