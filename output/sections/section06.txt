2-6 コンデンサ回路

【コンデンサの直列接続】

直列接続は，蓄えられる電荷が等しい
図2-37のように，2つ以上のコンデンサを直列につなぐ方法を直列接続といいます。この回路でコンデンサを充電すると，C₁ のコンデンサの正極から電子が C₂ のコンデンサの負極に移動したとみなすことができます。このとき，コンデンサの接続部分も静電誘導により電荷の移動が起こり，結果として2つのコンデンサに蓄えられる電荷は等しくなります。

（図2-37 直列接続は蓄えられる電荷が等しい）C₁〔+Q,−Q〕, V₁〔V〕／ C₂〔+Q,−Q〕, V₂〔V〕／電源 V〔V〕

直列接続の合成静電容量
図2-37のそれぞれのコンデンサにおいて Q=CV の関係が成り立つので，蓄えられた電荷 Q〔C〕は等しくなります。そこで，各コンデンサの静電容量を C₁〔F〕，C₂〔F〕，極板間の電位差を V₁〔V〕，V₂〔V〕とすると，次の等式が成り立ちます。
Q = C₁V₁〔C〕 → V₁ = Q/C₁〔V〕
Q = C₂V₂〔C〕 → V₂ = Q/C₂〔V〕

2つの極板間電位差 V₁ と V₂ の和が電源電圧 V と等しくなったときに充電が終わりますので，充電完了時では次の等式が成り立ちます。
V = V₁ + V₂ = Q/C₁ + Q/C₂ = Q(1/C₁ + 1/C₂)

=== p.134〜135 ===

したがって，直列接続された2つのコンデンサをまとめて合成静電容量 C をもつ1つのコンデンサとしてみなしたとき，Q=CV という関係がありますので，合成静電容量は次の式で求めることができます。
1/C = 1/C₁ + 1/C₂ → C = 1/(1/C₁ + 1/C₂)
なお，上式は
C = 1/(1/C₁ + 1/C₂) = C₁C₂/(C₁+C₂)
と「和分の積」の形に変形できます。この形の式は多用しますので，そのまま覚えてしまってもよいでしょう。また，静電容量が C₁，C₂，…，Cn の n 個のコンデンサを直列接続したときの合成静電容量 C〔F〕は，次の式で求めることができます。
C = 1/(1/C₁ + 1/C₂ + … + 1/Cn)〔F〕

公式：直列接続の合成静電容量
C = 1/(1/C₁ + 1/C₂ + … + 1/Cn)
C：直列接続の合成静電容量〔F〕，C₁, C₂, C₃：個々の静電容量 C₁, C₂,…, Cn〔F〕

（図：直列接続の合成静電容量）各静電容量の逆数の和
ふきだし：直列接続の合成静電容量は「各静電容量の逆数の和の逆数」と等しい！

メモチェ：
【コンデンサの直列接続】直列接続されたコンデンサに蓄えられる電荷はそれぞれ等しく，合成静電容量は各静電容量の逆数の和を，さらに逆数にしたものに等しい。

極板面積が等しく極板間距離が異なるコンデンサの直列接続
図2-38のように極板の面積 S〔m²〕が等しく，極板間距離がそれぞれ d₁〔m〕，d₂〔m〕の2つのコンデンサを真空中で直列接続した場合を考えます。

（図2-38 極板面積が等しく極板間距離が異なるコンデンサの直列接続）S〔m²〕, d₁〔m〕／S〔m²〕, d₂〔m〕

真空の誘電率を ε₀〔F/m〕とすると，各コンデンサの静電容量 C₁〔F〕，C₂〔F〕は次式で表されます。
C₁ = ε₀S/d₁〔F〕　C₂ = ε₀S/d₂〔F〕
コンデンサの直列接続の公式より，図2-38における合成静電容量 C〔F〕は
1/C = 1/C₁ + 1/C₂ = 1/(ε₀S/d₁) + 1/(ε₀S/d₂) = d₁/(ε₀S) + d₂/(ε₀S) = (d₁+d₂)/(ε₀S)
∴ C = ε₀S/(d₁+d₂)〔F〕

上式からわかるように，合成静電容量の極板間距離は，各コンデンサの極板間距離を足し合わせたものに等しくなります。直列接続の場合は，合成後のコンデンサはそれぞれのコンデンサの極板間距離の長さの分だけ長くなったと考えるとわかりやすいでしょう。

=== p.136〜137 ===

（図2-39 極板面積が等しい場合の直列接続）S〔m²〕, d₁, d₂ → S〔m²〕, d₁+d₂

【コンデンサの並列接続】

並列接続は，加わる電圧が等しい→極板間の電位差が等しい
図2-40のように，2つ以上のコンデンサを並列につなぐ方法を並列接続といいます。この回路では，各コンデンサにはそれぞれ同じ電圧が加わります。したがって，充電状態ではコンデンサ極板間の電位差が等しくなります。

（図2-40 並列接続では各コンデンサ極板間の電位差が等しい）印加される電圧は等しいが，蓄えられる電荷は各コンデンサの静電容量によって異なる／+Q₁,−Q₁（C₁）／+Q₂,−Q₂（C₂）／V〔V〕（3箇所同電圧）

並列接続の合成静電容量
図2-40において，それぞれのコンデンサには Q=CV の関係が成り立つので，極板間の電位差は等しく V〔V〕，各コンデンサの静電容量を C₁〔F〕，C₂〔F〕とし，蓄えられる電荷をそれぞれ Q₁〔C〕，Q₂〔C〕とすると，次の等式が成り立ちます。
Q₁ = C₁V〔C〕
Q₂ = C₂V〔C〕
並列接続されたコンデンサをまとめて1つの合成静電容量 C〔F〕をもったコンデンサとみなして，V〔V〕の電圧を加えて充電したとき，このコンデンサで蓄えられた電荷 Q〔C〕は Q=CV〔C〕となります。この Q は各コンデンサで充電した電荷量の和 Q₁+Q₂ と等しくなるので，次の等式が成り立ちます。
Q = CV = Q₁ + Q₂ = C₁V + C₂V = (C₁+C₂)V〔C〕
したがって，CV=(C₁+C₂)V という等式から C=C₁+C₂ となり，並列接続したコンデンサの合成静電容量は各コンデンサの静電容量の和に等しくなります。この関係は，コンデンサの数が増えて n 個になっても成り立ちます。

公式：並列接続の合成静電容量
C = C₁ + C₂ + … + Cn
C：並列接続の合成静電容量〔F〕，C₁, C₂,…, Cn：個々の静電容量〔F〕

（図：並列接続したコンデンサの合成静電容量）各静電容量の和
ふきだし：直流回路において抵抗の合成は，直列接続では単純な和で，並列接続では逆数の和の逆数です！だから，抵抗とコンデンサでは合成するときの式の形が逆になると覚えておきましょう！

メモチェ：
【コンデンサの並列接続】並列接続されたコンデンサは極板間の電位差が等しく，合成静電容量は各静電容量の和になる。

極板間距離が等しく，極板面積が異なるコンデンサの並列接続
図2-41のように極板間距離 d〔m〕が等しく，極板面積がそれぞれ S₁〔m²〕，S₂〔m²〕の2つのコンデンサを真空中で並列接続した場合を考えます。

=== p.138〜139 ===

（図2-41 極板間距離が等しい場合の並列接続）S₁〔m²〕, d〔m〕／S₂〔m²〕, d〔m〕

真空の誘電率を ε₀〔F/m〕とすると，各コンデンサの静電容量 C₁〔F〕，C₂〔F〕は次式で表されます。
C₁ = ε₀·S₁/d〔F〕　C₂ = ε₀·S₂/d〔F〕
コンデンサの並列接続の公式より，図2-41における合成静電容量 C〔F〕は
C = C₁ + C₂ = ε₀S₁/d + ε₀S₂/d = ε₀(S₁+S₂)/d〔F〕

上式からわかるように，各コンデンサの極板面積を足し合わせたものが合成後のコンデンサの極板面積になっているのと等価といえます。したがって，極板間距離が等しい場合の並列接続したコンデンサは，合成するとそれぞれの面積の分だけ広くなると考えるとわかりやすいでしょう。

（図2-42 極板間距離が等しい場合の並列接続）S₁+S₂〔m²〕, d〔m〕

【コンデンサの接続の応用①】

これまでの学習から，極板面積が等しく極板間距離が異なるコンデンサを直列接続すると，それらの距離の値を足し合わせた極板間距離をもつ1つのコンデンサに合成できる一方，極板間距離が等しく極板面積が異なるコンデンサの並列接続は，それらの極板面積の値を足し合わせた極板面積をもつ1つのコンデンサに合成できることがわかりました。ということは当然逆の操作も可能なので，図2-43のように1つのコンデンサを直列，もしくは並列に分割することができます。

（図2-43 コンデンサの直列分解と並列分解）S₁+S₂=S（並列）／d₁+d₂=d（直列）

【コンデンサの接続の応用②】

図2-44のように互いに d〔m〕の空げき間隔をおいて，平行平板状に並べられた11枚の同一形状の金属板を1枚おきに接続することで，コンデンサを形成することができます。

（図2-44 特殊な形のコンデンサ）極板が11枚。1枚おきに上下へ接続。

これと同じ構造で，極板を3枚に減らして電源を接続するとどのように充電されるかをみてみましょう。
図2-45（a）の上下の極板は電源の正極につながっていますので，正電荷が蓄えられる一方，図2-45（a）の中央の極板には負電荷が蓄えられます。このとき，上下の極板それぞれに蓄えられた電荷に対して，中央の極板にはその2倍の量の電荷が蓄えられます。
そうすると，図2-45（a）の中央の極板が図2-45（b）のように「2つの極板が一体になったものと等価である」とみなすことができるので，図2-45（c）のように回路を変形していくことができます。

=== p.140〜141 ===

（図2-45 3枚の極板によるコンデンサ）（a）（b）（c）

結果として，3枚の極板によりつくるコンデンサは，2つのコンデンサを並列に接続したものと等価となります。極板が5枚であれば4つのコンデンサ，7枚であれば6つのコンデンサと，「（極板の枚数）−1 の数のコンデンサの並列接続」としてみなすことができるため，図2-46のように少ない極板の枚数で大きな静電容量のコンデンサを構成することができます。

（図2-46 並列接続されたコンデンサと等価）

このような構造のコンデンサは一般に積層コンデンサといい，主に積層セラミックコンデンサや可変コンデンサに応用されています。

（図2-47 積層コンデンサの応用）（a）積層セラミックコンデンサ：外部電極・内部電極・誘電体（b）可変コンデンサ（バリコン）

メモチェ：
【積層コンデンサ】極板を互い違いに配置することで，少ない極板の枚数で並列接続されたコンデンサを形成することができる。

【コンデンサの分圧則】

直流回路の章にて抵抗の分圧則を学習しましたが，コンデンサが直列接続されたときの分圧にも法則があり，その分圧式は次のように表すことができます。

（図：直列接続におけるコンデンサの分圧式（コンデンサが2つの場合））
C₁──V₁→、C₂──V₂→、V→
V₁ = C₂/(C₁+C₂)·V
V₂ = C₁/(C₁+C₂)·V
ふきだし：コンデンサの分圧式は，「並列接続された抵抗の分流則」と同じ形っす！

上図のように静電容量が C₁〔F〕，C₂〔F〕の2つのコンデンサが直列接続されるとき，蓄えられる電荷 Q〔C〕は各コンデンサで等しく，各コンデンサに加わる電圧をそれぞれ V₁〔V〕，V₂〔V〕とすると，電荷 Q〔C〕は次の式で求めることができます。
Q = C₁V₁〔C〕　Q = C₂V₂〔C〕

=== p.142〜143 ===

また，2つのコンデンサの合成静電容量を C〔F〕，電源電圧を V〔V〕とすると，蓄えられる電荷 Q〔C〕は次の式で求めることができます。
Q = CV = C₁C₂/(C₁+C₂)·V〔C〕
各式を等号で結ぶことで，各コンデンサの極板間の電位差を次のように求めることができ，その形は，直流回路の章で学習した「並列接続された抵抗の分流の式」と同じ形になります。
C₁V₁ = C₁C₂/(C₁+C₂)·V → V₁ = C₂/(C₁+C₂)·V〔V〕
C₂V₂ = C₁C₂/(C₁+C₂)·V → V₂ = C₁/(C₁+C₂)·V〔V〕

問題15（H28 問7）
静電容量が 1 μF のコンデンサ3個を下図のように接続した回路を考える。すべてのコンデンサの電圧を 500 V 以下とするため，a-b 間に加えることができる最大の電圧 Vm の値〔V〕として，最も近いものを次の（1）〜（5）のうちから一つ選べ。ただし，各コンデンサの初期電荷は零とする。
（図：a-C-（並列2個）-b, 両端に Vm〔V〕）

（1）500　（2）625　（3）750　（4）875　（5）1000

NOTE：分圧則を理解していれば素早く解ける問題ですが，理解していない場合かなり遠回りになります。

解説 まず，コンデンサの並列部分に着目すると，これの合成静電容量は 2 μF となります。したがって，題意の回路は解図のような2つのコンデンサの直列接続したものと等価となります。
（解図：a ── C=1μF ── C'=2μF ── b，V₁, V₂, Vm）

ここで，電圧 Vm〔V〕を印加したときに各コンデンサに加わる電圧は V₁〔V〕，V₂〔V〕は，コンデンサの分圧則より
V₁ = C'/(C+C')·Vm = (2/3)Vm〔V〕
V₂ = C'/(C+C')·Vm = (1/3)Vm〔V〕
　※原本表記。V₂ は正しくは C/(C+C')·Vm＝(1/3)Vm であり，分子の記号の書き間違いがある可能性

となるので，解図の左のコンデンサ C に加わる電圧の方が高くなります。この電圧が 500 V 以下になれば題意を満たすため
V₁ = (2/3)Vm ≦ 500
∴ Vm ≦ 500×(3/2) = 750 V

問題16（H26 問5）
図のように，コンデンサ3個を充電する回路がある。スイッチ S₁ 及び S₂ を同時に閉じてから十分に時間が経過，定常状態となったとき，a 点からみた b 点の電圧の値 V〔V〕として，正しいものを次の（1）〜（5）から一つ選べ。ただし，各コンデンサの初期電荷は零とする。
（図：S₂-20V, a-20μF-b, S₁-10V, コンデンサ 10μF が2個）

（1）−10/3　（2）−2.5　（3）2.5　（4）10/3　（5）20/3

=== p.144〜145 ===

解説 まず，求めるのは a 点からみた b 点の電位なので，解図1の矢印の向きを正として Vba〔V〕を定義します。
電源が2つあるため，重ね合わせの理を適用します。20 V の電源が単独で存在する回路を解図2（a），10 V の電源が単独で存在する回路を解図2（b）に示します。このとき，C₂ の端子電圧をそれぞれ同図の向きに V₁〔V〕，V₂〔V〕とします。
解図2（a），（b）において，並列部分のコンデンサを合成した回路をそれぞれ解図3（a），（b）に示します。

NOTE：直流回路の章で学習した重ね合わせの理をコンデンサ回路に応用した問題です。

解図3（a）において，V₁ は合成したコンデンサ C₂₃ の分担電圧なので，コンデンサの分圧則より
V₁ = C₁/(C₁+C₂₃)×20 = (10×10⁻⁶)/(10×10⁻⁶ + 30×10⁻⁶)×20 = 5.0 V
となります。一方解図2（b）において V₂ は合成したコンデンサ C₁₂ の分担電圧なので，分圧則より
V₂ = C₃/(C₁₂+C₃) = (10×10⁻⁶)/(10×10⁻⁶ + 30×10⁻⁶)×10 = 2.5 V

以上より，これらの結果を重ね合わせると解図4のようになります。よって，電圧 Vba は，その向きに注意すると
Vba = V₁ − V₂ = 5.0 − 2.5 = 2.5 V
