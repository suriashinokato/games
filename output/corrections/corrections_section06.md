### #01 — p.143 問題15 解説の分圧式 V₂ の分子記号

- 該当箇所：問題15（H28 問7）解説における V₁、V₂ の分圧式
- 抽出テキスト上の表記：
  ```
  V₁ = C'/(C+C')·Vm = (2/3)Vm
  V₂ = C'/(C+C')·Vm = (1/3)Vm   ← ※分子がC'のまま
  ```
- 正しいべき表記（数値と整合する形）：
  ```
  V₁ = C'/(C+C')·Vm = (2/3)Vm     ← C = 1μF、C' = 2μF
  V₂ = C/(C+C')·Vm  = (1/3)Vm     ← 分子は C（大文字プライムなし）
  ```
- 理由：C = 1μF、C' = 2μF なので、V₂ が (1/3)Vm となるためには分子が C（=1μF）でなければならない。もし分子が C'（=2μF）なら V₂ = 2/3·Vm となり、(1/3)Vm と矛盾する。
- 論点：**PDF 紙面上は「C」と「C'」が小さな違いで組版されており、テキスト抽出時に両方「C'」として拾われている可能性が高い**。画像上は正しく `V₂ = C/(C+C')·Vm` と書かれていると推定されるが、**紙面を拡大して目視確認**してください。
