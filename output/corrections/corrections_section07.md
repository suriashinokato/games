### #01 — p.156 電荷量保存の法則の説明「全体の電荷の総量 2Q〔C〕から…」

- 該当箇所：p.156「電荷量保存の法則」の導入部分（図2-55 直前の本文）
- 誤：
  > 「電荷量保存の法則から、接触する前と接触したあとで全体の総量は変化しないので、接触した後の B の電荷は**全体の電荷の総量 2Q〔C〕**から、A がもっている電荷 2Q〔C〕を引いた残りの量である Q〔C〕となります。」
- 正：
  > 「電荷量保存の法則から、接触する前と接触したあとで全体の総量は変化しないので、接触した後の B の電荷は**全体の電荷の総量 3Q〔C〕**から、A がもっている電荷 2Q〔C〕を引いた残りの量である Q〔C〕となります。」
- 理由：
  - 接触**前**：A=3Q〔C〕、B=0 → 合計 **3Q**
  - 電荷量保存より接触**後**も合計 **3Q**
  - 接触後：A=2Q に減少 → B = 3Q − 2Q = Q ✓
  - 文中の「全体の電荷の総量 2Q〔C〕」は A の接触後の電荷量と同値であり、この値から A の電荷（2Q）を引くと 0 になってしまい、「残りの量である Q〔C〕」と結論が合わない。明らかに **3Q** の書き間違い。
- 種別：[原本の誤記]
- 観点：数式記号の整合性／本文ロジック
- 重要度：★★★（電荷量保存の法則を初めて学ぶ学習者にとって、導入部のこの計算が合わないと本節全体の理解に致命的な混乱を招く）

### #02 — p.161 問題20（b）解説「CB0 = (1/3+1+3)ε₀S/s」

- 該当箇所：問題20（R3 問17）（b）解説の合成静電容量 CB0 の途中式
- 誤：
  > CB0 = CB1 + CB2 + CB3 = ε₀S/(3d) + ε₀S/d + 3ε₀S/d = (1/3+1+3)**ε₀S/s** = (13/3)·ε₀S/d
- 正：
  > CB0 = CB1 + CB2 + CB3 = ε₀S/(3d) + ε₀S/d + 3ε₀S/d = (1/3+1+3)**ε₀S/d** = (13/3)·ε₀S/d
- 理由：同じ計算式の途中と直後で、分母の記号が「/s」と「/d」で食い違っている。本問は極板間距離 d で統一されており、「s」という記号は本問に登場しない。明らかに「d」の誤植（キー隣接ミス）。
- 種別：[原本の誤記]
- 観点：単位・記号の誤記
- 重要度：★★（計算結果 (13/3)·ε₀S/d は正しいので、最終解答の 0.83 倍には影響しないが、途中式の誤植は修正すべき）
