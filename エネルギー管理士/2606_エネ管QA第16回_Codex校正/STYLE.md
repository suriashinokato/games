# STYLE

対象: `【初校】2606_お悩み解決！ エネ管Q&A_第16回.pdf`

## 文体

- 読者に語りかけるQ&A調。会話文のくだけた表現は許容。
- 解説本文は「です・ます」調で統一。
- 初学者向けのため、断定が強すぎる箇所は「得点につなげやすい」などに弱める余地あり。

## 表記

- 数字: 見出し・本文とも全角数字が多い。式中・記号中は半角英数字。
- 単位: `［V］`、`［Ω］`、`［A］`のように全角角括弧。
- 図番号: Qごとに `図1` から始まっている。雑誌側ルールとしてQ単位リセットなら問題なし。
- 用語: 情報処理系は「漏えい」「ウイルス」への統一を推奨。
- 英語併記: 重要語の初出付近に括弧付きで併記。

## 数式・記号

- 交流回路の電圧・電流はドット付きフェーザ表記。
- `jX1`、`jX2`、`-jX3` は下付き数字で組まれている。
- 抽出テキストではドット、下付き、分数が崩れるため、数式判断はページ画像を基準にした。
