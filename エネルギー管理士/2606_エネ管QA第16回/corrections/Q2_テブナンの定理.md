# Q2 校正結果 — 「交流回路でのテブナンの定理の適用方法を教えてください！」

- 対象: p.102–103（2 ページ、2 段組）
- 課目区分: Ⅱ電気の基礎 — 電気理論
- 担当キャラ: 電気男

## 1. 全体所見

直流回路でのテブナンの定理の復習から、交流回路（複素インピーダンス＋フェーザ）への適用までを段階的に提示しており、構成・記法ともに非常に良い。**Ė, V̇₀, Ż₀, İ_R, V̇_R のフェーザ表記（ドット）も全式で一貫**。すべての導出式を独立検算で再現した結果、**計算誤りはなし**。鳳・テブナンの定理という日本での慣習名も正しい。指摘は表現レベルの微細な気付きのみ。

## 2. 計算検証（独立検算）

### (1) 開放電圧 V̇₀
原稿:
$$\dot{V}_0 = \frac{-jX_3}{jX_1 - jX_3}\dot{E} = -\frac{X_3}{X_1 - X_3}\dot{E}\ [\mathrm{V}]$$

**検算**: 抵抗 R を取り除くと a–b は開放、jX₂ には電流が流れない。よって a–b 端子電圧は −jX₃ の両端電圧に等しい。Ė は jX₁ と −jX₃ の直列に印加されているので分圧で
$$\dot{V}_0 = \frac{-jX_3}{jX_1 + (-jX_3)}\dot{E} = \frac{-jX_3}{j(X_1 - X_3)}\dot{E} = -\frac{X_3}{X_1 - X_3}\dot{E}$$
**→ 一致 ✅**

### (2) 等価インピーダンス Ż₀
原稿:
$$\dot{Z}_0 = jX_2 + \frac{jX_1 \cdot (-jX_3)}{jX_1 - jX_3} = j\!\left(X_2 - \frac{X_1 X_3}{X_1 - X_3}\right)\ [\Omega]$$

**検算**: 電源短絡後、a–b から見て jX₂ が直列、その奥で jX₁ ∥ (−jX₃)。並列合成は
$$\frac{jX_1 \cdot (-jX_3)}{jX_1 + (-jX_3)} = \frac{X_1 X_3}{j(X_1 - X_3)} = -\frac{j X_1 X_3}{X_1 - X_3}$$
これに jX₂ を加えると
$$\dot{Z}_0 = jX_2 - \frac{jX_1 X_3}{X_1 - X_3} = j\!\left(X_2 - \frac{X_1 X_3}{X_1 - X_3}\right)$$
**→ 一致 ✅**

### (3) İ_R, V̇_R
原稿:
$$\dot{I}_R = \frac{\dot{E}_0}{\dot{Z}_0 + R} = \frac{-X_3}{R(X_1 - X_3) + j(X_1 X_2 - X_2 X_3 - X_3 X_1)}\dot{E}\ [\mathrm{A}]$$
$$\dot{V}_R = R\dot{I}_R = \frac{-RX_3}{R(X_1 - X_3) + j(X_1 X_2 - X_2 X_3 - X_3 X_1)}\dot{E}\ [\mathrm{V}]$$

**検算**: 分母を (X₁ − X₃) で通分し直す。
$$\dot{Z}_0 + R = \frac{j\{X_2(X_1-X_3) - X_1 X_3\} + R(X_1-X_3)}{X_1 - X_3}$$
$$\dot{I}_R = \frac{-\dfrac{X_3}{X_1-X_3}\dot{E}}{\dfrac{j\{X_2(X_1-X_3) - X_1 X_3\} + R(X_1-X_3)}{X_1-X_3}} = \frac{-X_3 \dot{E}}{R(X_1-X_3) + j\{X_1 X_2 - X_2 X_3 - X_1 X_3\}}$$
ここで X₃X₁ = X₁X₃ なので原稿の j(X₁X₂ − X₂X₃ − **X₃X₁**) と一致。
**→ 一致 ✅**

V̇_R = R · İ_R も自動的に正しい。**→ 一致 ✅**

## 4. 🟢 軽微な気付き（いずれも修正必須ではない）

### #Q2-1 「並列合成インピーダンス」の和の表記　【観点：表現】
- p.103 (2): 「jX₁ および −jX₃ の **並列合成インピーダンス** を、さらに直列合成したものに等しくなるから」のあとの式が
  `jX₁·(−jX₃) / (jX₁ − jX₃)`
  となっている。これは `(jX₁) + (−jX₃) = jX₁ − jX₃` の和を分母に書き下した形であり、計算的に正しい。ただし読者によっては「並列の和なのに引き算？」と一瞬迷う恐れがあるため、必要なら式の直前または直後に
  「（分母 jX₁ − jX₃ は (jX₁) + (−jX₃) であることに注意）」
  といった注記を 1 行加えると親切。**判断**: 任意。

### #Q2-2 İ_R の分母虚部 X₃X₁ の表記順　【観点：表記の好み】
- 原稿: `j(X₁X₂ − X₂X₃ − X₃X₁)`
- 一貫性の観点では `j(X₁X₂ − X₂X₃ − X₁X₃)` のほうが、添字昇順で揃って読みやすい（X₃X₁ だけ降順）。
- 数学的に等価。**判断**: 編集の好みの範囲。修正不要レベル。

### #Q2-3 「電圧 Ė[V] の定電圧源」　【観点：用語】
- 「定電圧源」は理想電圧源（内部インピーダンス 0）を強調する用語。本問でも内部インピーダンスを考えていない前提なので妥当。エネ管出題でも「定電圧源」表現は用例あり。
- 「電圧源」だけでも通じるが、原稿のままで OK。**修正不要**。
